Avaturn ‹iframe› example

  1. Click Open Avaturn button.
  2. In the avatar editor click SAVE button when you are finished customizing your avatar.
  3. The parent page (this page) will receive a raw glb file in DataURL format. You can then convert it to raw bytes or create an ObjectURL.
  4. For illustrative purposes a link to download avatar will appear so you can see the file has actually been received.
  5. Close Avaturn window with Close Avaturn
